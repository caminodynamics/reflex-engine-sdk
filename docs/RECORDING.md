# Recording the Demo

## First-Time Build

```bash
cargo build --release --bin demo
```

## Recording Command

```bash
run-demo.bat
```

## Expected Output

The demo will create audit artifacts in:
```
./artifacts/evt-001.json
./artifacts/evt-002.json
```

## Recording Tips

- **Terminal size:** 80x24 characters minimum
- **Font:** Monospace (Consolas, Monaco, Fira Code)
- **Window:** No transparency, solid background
- **Timing:** Demo completes in ~2 seconds
